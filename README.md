## How do I install these formulae?

`brew install czchen/repo/<formula>`

Or `brew tap czchen/repo` and then `brew install <formula>`.

Or, in a [`brew bundle`](https://github.com/Homebrew/homebrew-bundle) `Brewfile`:

```ruby
tap "czchen/repo"
brew "<formula>"
```
